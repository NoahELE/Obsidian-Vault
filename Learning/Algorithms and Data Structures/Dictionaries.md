# Dictionaries

Implementation below:

- Static Array
    - Insert: $O(1)$
    - Search: $O(n)$
- Dynamic Array
    - Insert: $O(n)$
    - Search: $O(n)$
    - No limit to key/values & efficient space
- Sorted Dynamic Arrays
    - Insert: $O(n)$
    - Search: $O(logn)$
- Linked list
    - Insert: $O(1)$
    - Search: $O(n)$
    - No limit to key/values & efficient space
