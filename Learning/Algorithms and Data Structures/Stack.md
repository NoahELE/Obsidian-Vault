# Stack

- Insert at top (newest)
- Take from top (newest)
- Abstract Data Structure
  - Push – Add an item to the top of the stack
  - Pop – Take an item from the top of the stack

- Linked List
  - Push - $O(1)$ - Prepend
  - Pop - $O(1)$ - `top = top->next`
- Dynamic Array
  - Push - $O(n)$ - $O(n)$ if we have to realloc
  - Pop - $O(1)$
